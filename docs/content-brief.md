- 品牌名：MIYARISAN 妙利散（日本 Miyarisan Pharmaceutical Co., Ltd. 製造）

- 代理商：R-CHEVER
- 核心口號：**全員抵達腸道 發揮作用**
- 定位：**次世代益生菌製劑**，治療範圍超越胃腸科

## 產品規格

- 劑型：整腸劑 活性宮入菌（芽孢菌）製劑
- 包裝：1g/包 × 42包
- 每包含 CBM 588：30mg
- 主要成分：**酪酸**（Butyric Acid）

## 好的益生菌 5 條件（Fuller, 1989）

| 條件 | 說明 |
|------|------|
| 活菌 | 必須是活菌才能抑制疾病，對人體或宿主有益 |
| 無毒性 | 不會致病且無毒性，對人體無任何傷害 |
| 耐胃酸 | 能夠通過胃酸，穩定於腸道存活、代謝並發揮功效 |
| 安定性強 | 長期貯存安定性強，功效不受影響 |
| 基因穩定 | 遺傳基因穩定，功效不因遺傳而降低效用 |

## 宮入菌 vs. 乳酸菌 關鍵差異

| 項目 | 宮入菌（妙利散） | 乳酸菌 |
|------|----------------|--------|
| 耐酸 | 耐酸 pH1.4，活菌型態通過胃抵達腸道 | 不耐酸，pH3.5 以下幾乎不能生存 |
| 耐熱 | 耐熱達攝氏 55°C | 不耐熱，低溫存放 |
| 抗生素 | 可與抗生素併用 | 幾乎不可與抗生素併用 |
| 保存 | 可室溫保存 | 需低溫保存 |
| 分泌物 | 酪酸、乳酸、乙酸 | 僅乳酸 |
| 特殊功能 | 增加體內 A 菌 B 菌繁殖、具酪酸功能 | 不具酪酸功能 |
| 過敏 | 牛奶蛋白過敏患者適用 | 可能引發腹瀉等過敏症狀 |
| 嬰幼兒 | 能加入牛奶給寶寶食用 | — |

## 宮入菌特色：芽孢

- 對熱和乾燥具有抵抗力
- 不受胃酸和膽汁酸影響，以活菌狀態通過胃抵達腸道
- 不受抗生素影響，可與抗生素併用

---

## 作用機轉

1. 抗菌物質之生長
2. 短鏈脂肪酸之產生
3. 競合的排除
4. 破壞毒素接受體
5. 促進 IgA 之分泌
6. 增強腸黏膜屏障
7. 群體感應
8. 抑制毒素產生
9. 免疫控制

## 酪酸（Butyric Acid）作用

**腸道直接效益：**
- 止瀉解便秘
- 修復潰瘍
- 抗發炎症狀

**六大機制：**
- 保護大腦避免低度發炎
- 促進腸神經系統之健康
- 調節腸黏膜免疫功能
- 提供腸壁細胞能源
- 改善腸管血液循環和蠕動
- 避免腸道發炎和結腸癌

---

## 新世代應用範疇

### 調節免疫
- 促進 T-reg，抑制腸道炎症及過敏反應
- 應用：過敏、異位性皮膚炎、過敏性鼻炎
- 機制：TGF-β 誘導 Regulatory T-Cell（Treg）形成，調節 IgE 分泌

### 腸腦軸線（Gut-Brain Axis）
- 宮入菌 CBM588 可在「腸－腦軸線」循環中發揮功效
- 應用：憂鬱症、焦慮症、慢性壓力治療

---

## 應用疾病範圍

**消化道：**
病毒感染腹瀉、抗生素腹瀉、旅行性腹瀉、發炎性腹瀉、大腸癌內（含潰瘍性大腸炎、幽門螺旋菌感染、困難梭桿菌感染）、輔助治療、腸躁症、大腸結腸癌

**非消化道：**
- 免疫系統：增強免疫力、類風濕性關節炎
- 泌尿生殖：細菌、白色念珠菌陰道炎
- 皮膚：異位性皮膚炎、濕疹
- 精神疾病：憂鬱
- 肝臟：膽囊結石、非酒精性脂肪肝、非酒精性肝硬化
- 肺臟：延長非小細胞肺癌患者生存期及復發緩解期

## 臨床數據

### 對消化疾病（IBD）效用
治癒率 **77.8%**（499/641 人）

| 症狀 | 治療前 | 治療後 |
|------|--------|--------|
| 腸胃炎 | 75% | 顯著降低 |
| 腹部症狀 | 80% | 顯著降低 |
| 腹瀉 | 97% | 顯著降低 |
| 便秘 | 67% | 顯著降低 |
| 交替性通便異常 | 80% | 顯著降低 |
| 便溏 | 59% | 顯著降低 |

### 對抗生素輔助（腹瀉發生率）
- 單純抗生素：47.5%
- 抗生素 + Miyarisan BM 1g：28.0%
- 抗生素 + Miyarisan BM 3g：**2.9%**

### 對 IBS 效用（123 人）
整體改善率 **80.5%**，其中腹脹改善 87.5%、腹瀉 86.4%

### 對嬰幼兒（60人，5天～8歲）
- 腹瀉療效 **89.3%**
- 呼吸道感染改善 **100%**

## 抗生素耐受性

宮入菌對各類代表性抗生素之耐受性均經追蹤實驗確認，從 1930 年代到 2020 年各世代抗生素（含 Quinolone 類、β-內醯胺類抗生素、第五代 Ceftaroline、新型抗生素等）均可與宮入菌並用，且不會有不良反應。
