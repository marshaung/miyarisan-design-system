# 妙利散 Website UI Kit

## Overview
Interactive recreation of the Miyarisan Taiwan website — a medical-grade probiotic information and sales site targeting physicians and pharmacists.

## Design Source
- `marshaung/miyarisan-web` — `index.html` (full production single-page app)
- Screenshots: `assets/homepage-draft.jpg` (pink theme), `assets/homepage-dark.png` (navy dark theme)

## Screens / Components
- **Nav** — frosted glass (light) + dark navy variant; sticky; tab bar
- **Hero** — two variants: dark navy (primary) + hot-pink gradient
- **Stats Bar** — 4-col key metrics
- **Pillar Cards** — 3-col "medical grade conditions"
- **Feature Cards** — 2-col split (acid resistance + antibiotic use)
- **Comparison Table** — CBM588 vs. general probiotics
- **Research Stats + Cards** — clinical data cards with bar charts
- **FAQ Accordion** — expandable Q&A
- **Footer** — dark navy + brand info

## Design Width
1280px desktop (responsive down to 375px mobile)
